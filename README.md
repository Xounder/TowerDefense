
# TowerDefense
A Tower Defense made in Python (Pygame)

# EN
I was without internet for a week and out of boredom I ended up having the will/idea to do a Tower Defense, well, here it is hehe

- About the game:

This is a Tower Defense game (as the name says defend the tower), so with each Wave you will face enemies trying to protect your tower with each Wave of enemies.

By defeating enemies, you will earn gold that you will use to improve your protection or recover them.

With each wave, the number of enemies will increase and different species will appear, which have their own characteristics (more speed, more life...)

In certain waves a Boss and a MiniBoss will appear, by defeating them you will receive a relevant amount of gold
 
- Instructions:

      Tab - Use during the shopping phase to know the value of each item (complete)
      Esc - Use during waves to pause the game
      Mouse/Space - Shoot
      Mouse - Perform actions within the game
      Q - Use Missile
      E - Use the Item that Heals the Tower and makes it Invulnerable

# PT/BR
Fiquei uma semana sem internet e no tédio acabei tendo a vontade/ideia de fazer um Tower Defense, pois bem, aki está hehe

- Sobre o Jogo:

Este é um jogo Tower Defense (como o proprio nome diz defender a torre), então a cada Wave (onda) você enfrentará inimigos tentando proteger a sua torre a cada Wave de inimigos.

Ao derrotar imigos ganhará gold que será utilizado para melhorar sua proteção ou recuperá-las.

A cada wave a quantidade de inimigos auumentará e aparecerá espécies diferentes, que possuem suas próprias caracteristicas (mais velocidade, mais vida ...)

Em certas waves aparecerá um Boss e MiniBoss, ao derrotá-los receberá uma quantidade relevante de gold
 
- Instruções:

		Tab - Use durante a fase de compras para saber o valor de cada item (completo)
		Esc - Use durante as waves para pausar o jogo
		Mouse/Space - Atira
		Mouse - Realiza as ações dentro do jogo
		Q - Usa o Missil
		E - Usa o Item que Cura a Torre e a deixa Invulneravel
